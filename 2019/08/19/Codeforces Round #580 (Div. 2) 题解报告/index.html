A

You are given an array A, consisting of n positive integers a1,a2,…,an, and an array B, consisting of m positive integers b1,b2,…,bm.

Choose some element a of A and some element b of B such that a+b doesn’t belong to A and doesn’t belong to B.

For example, if A=[2,1,7] and B=[1,3,4], we can choose 1 from A and 4 from B, as number 5=1+4 doesn’t belong to A and doesn’t belong to B. However, we can’t choose 2 from A and 1 from B, as 3=2+1 belongs to B.

It can be shown that such a pair exists. If there are multiple answers, print any.

Choose and print any such two numbers.

题意大概就是给你一个数组A,B,让你求出某一个$A_i+B_i$不属于A和B的某一个解.

很显然就是傻逼签到题,直接开桶枚举就好了.

B

You are given nn numbers a1,a2,…,ana1,a2,…,an. With a cost of one coin you can perform the following operation:

Choose one of these numbers and add or subtract 11 from it.

In particular, we can apply this operation to the same number several times.

We want to make the product of all these numbers equal to 11, in other words, we want a1⋅a2a1⋅a2 …… ⋅an=1⋅an=1.

For example, for n=3n=3 and numbers [1,−3,0][1,−3,0] we can make product equal to 11 in 33 coins: add 11 to second element, add 11 to second element again, subtract 11 from third element, so that array becomes [1,−1,−1][1,−1,−1]. And 1⋅(−1)⋅(−1)=11⋅(−1)⋅(−1)=1.

What is the minimum cost we will have to pay to do that?

给你一个数列,你每一次可以选择一个数字,使他加1或者减1,求最少多少次能使所有数字相乘结果为1.

对于正数,显然使这个数变成1最优,但是要知道,只有偶数个-1时,答案才能-1.所以还需要处理偶数个-1.但是如果负数都变成-1之后有奇数个,就需要使ans+=2.
最后别忘了0的情况.

D

You are given n integer numbers a1,a2,…,an. Consider graph on n nodes, in which nodes i, j (i≠j) are connected if and only if, ai AND aj≠0, where AND denotes the bitwise AND operation.

Find the length of the shortest cycle in this graph or determine that it doesn’t have cycles at all.

给你一个数组A,如果A_i&A_j==true的话,那么两点有边,求最小环.

按照ai&(1ll<<i)进行分组,如果在某一次分组中出现了值大于等于3的情况,3就为答案.不然的话建边,再求最小环.可以证明边最多是64条,所以不怕TLE.
